Navigasi Galeri & Album

Gunakan menu di bawah (atau atas untuk komputer) untuk berpindah. Galeri berisi koleksi foto umum tanpa folder, sedangkan Album memuat foto-foto yang dikelompokkan berdasarkan acara/kegiatan.

Mengubah Tema Tampilan

Aplikasi ini mendukung Mode Terang (siang) dan Mode Gelap (malam). Klik tombol ikon Matahari/Bulan di pojok kanan atas layar untuk menyesuaikan kenyamanan mata Anda.

Melihat Foto Penuh

Klik sembarang foto di Mading, Bahan Baca, atau Galeri untuk memperbesarnya (mode Layar Penuh/Lightbox). Anda dapat membaca judul dan deskripsi lengkap dari foto tersebut.

Panduan Khusus Administrator

1

Sistem Keamanan Sandi

Semua tindakan pengelolaan data (Upload, Buat Folder, Bagikan Tautan, Edit, Hapus, dan Download) WAJIB menggunakan Sandi Administrator (60400173). Jaga kerahasiaan sandi ini sebaik mungkin.

2

Mengunggah (Upload) Foto Baru

Gunakan tombol Unggah Foto. Sistem otomatis akan mengompres foto berukuran besar menjadi format WebP sangat kecil (~30KB) tanpa merusak kualitas. Aplikasi dijamin mampu menampung 5.000+ foto dengan sangat cepat.

3

Membagikan Tautan Spesifik (Share)

Klik tombol biru Bagikan pada foto atau album tertentu. Masukkan sandi, lalu salin link tersebut ke WhatsApp. Orang yang menerima link hanya bisa "Melihat" saja (Read-Only).

Mengedit Judul & Deskripsi Foto

Klik tombol ikon pensil berwarna kuning emas di pojok setiap foto. Anda bisa mengedit atau memperbaiki keterangan foto dengan mudah. (Berlaku untuk foto *default* maupun foto *cloud*).

Menghapus Foto atau Folder

Klik tombol tempat sampah merah pada foto individual, atau tombol merah bertuliskan Hapus Semua di dalam folder.
*Peringatan Penting: Menghapus folder akan memusnahkan seluruh foto di dalamnya secara permanen dari server database Firebase!

Tanya Jawab (FAQ)

Mengapa butuh Sandi untuk Mendownload?

Demi mencegah penyalahgunaan atau pengunduhan massal oleh pihak luar, akses unduh (simpan) resolusi tinggi kini dikunci oleh sistem aplikasi.

Apakah foto yang dihapus bisa dikembalikan?

Tidak bisa. Aplikasi langsung terhubung (real-time) dengan Cloud Firestore. Data yang terhapus setelah validasi sandi akan lenyap selamanya.

Kenapa foto langsung muncul (tanpa loading)?

Aplikasi ini mematikan penundaan animasi (DOM Batching) dan mengubah ukuran fotonya saat *upload*. Ini memangkas waktu *loading* galeri hingga 90%!

Apakah memori HP saya akan penuh?

Sama sekali tidak. Aplikasi ini berbasis PWA (*Progressive Web App*). Seluruh fotonya tersimpan aman di server Cloud Google Firebase, bukan di dalam HP Anda.
